# 应用介绍
番茄钟是针对短时间工作的任务定制，而番茄周旨在为您一周的每一天定制工作任务，这样每天有个小目标，每周一个大目标，日积月累，您会发现您的工作是如此地有效率！

# 库使用

|    名称       |  用途         |
| ------------- |:-------------:|
| Realm         | 移动端数据库  |
| ButterKnife   |  视图注解     |
| ~~Fresco~~    | ~~图像加载~~  |
|Material-Ripple| 水波纹特效    |
| RxAndroid     | 响应式编程    |
| Glide         | 图片加载库    |
| Dagger2       | 依赖注入库    |
